
	Область видимости переменной.
		это откуда мы видим переменную, то откуда мы можем к ней обратиться.
		
	Цикл и условия не создают локальной видимости.
			for ($i = 0; $i < 3; $i++) {
				echo $i . "\n";
				$b = 'hello';
			}
			echo $i;			// Нет понятия внутри и вне цикла. $i выведется 3.
			echo $b; 			// hello
			
			if (true) {
				$value = 'cat';
			} else {
				$value = "";
			}
			echo $value;		// cat
		
	Функции создают локальную видимость.
			
		
	Чистая функция зависит только от своих локальных параметров и не 
	зависит от какого-либо внешнего состояние.
	 Не меняют внешние переменные.
	Одно из главных правил хорошего программирования не используйте глобал переменные.
		
		2способа работы с глоб переменной.
			$increment = 1;
			function increment2($num)
			{
				global $increment;
				return $num + $increment;
			}
			
			function increment3($num)
			{
				return $num + $GLOBALS['increment'];
			}

ВОПРОСЫ
	Глобальная переменная может быть изменена в любой точке программы 
	   (что может приводит к ошибкам в разных частях программы)
	Глобальные переменные создают сильные зависимости между разными участками кода, 
	   которые приводят к более сложным программам
	Глобальные переменные усложняют интеграцию модулей, потому что разные модули могут 
	     использовать одни и те же названия для глобальных переменных
