
	Символьные классы предназначены для задания набора символов, которым должна соответствовать подстрока
		
			[a-z]				// Все символы англ алфавита в нижнем регистре
			[0-9]				// Все цифры
			[aj]				// а строчные и j
			[^aj]				// отрицание. инвентирование. Все символы кроме a и j
			[aj-]				// строчные а j  и еще  -  дефис
			\d					// задается спец класс цифр (все цифры [0-9])
			\D					// все символы (буквы, пробелы и т.д.) кроме цифр
			\s					// все пробельные символы
			\S					// все НЕпробельные символы
			\w					// все буквы цифры и подчеркивания
			
		\w  равносильно  [0-9a-zA-Z_]	
			\W					// инвертированный вариант   \w

	Первый и второй символ это числа
	Третий символ это /
	Четвертый это любой символ за исключением a-z
