
	По умолчанию регулярки ищут подстрочки
	вот такому регулярному   java    	 соответствует   'asdfjava'  'java' 'tetjava4jf'
	
	
			^java  			//  Проверка делается только вначале строки   'java and javaScript'   // только первая java 
			script$			// Сопоставление только концу строки
			
			a\b				// Поиск символа  a  только в конце слова
			a\B				// происходит инверт. Соответсвие  все  а  но только не в конце слова
			
			\bj				// поиск  j  в начале слов
			\Bj				// Все   j  кроме тех, что вначале слов 
			\Bj\Bj			// Все   j  кроме тех что вначале и конце слов 
			
			
			
ТЕСТЫ
	Символ $ означает
		конец строки
		
	При обозначении позиции символ ^ означает
		начало строки
	
	Дана строка:
		hello and hello again, my friend!
			И регулярное выражение:
			/^hello\b/g
		Сколько совпадений будет найдено?
			1
			
	Дана строка:
		This is a nice day, but you are not going anywhere.
		Дано регулярное выражение:
			/^a/g
		Сколько совпадений будет найдено?
			0

							^support@hexlet\.io$
							END
