
	Цель: Познакомиться с системой прав в *nix системах. 
	
	В линукс системах существуует еще распределени по правам. кто какой доступ может иметь к файлам и папкам.
	
	
				ls -la		// вывод файлов текущей директории
				
		
		drwxrwxrwx 
		папка, владелец, группа, прочие пользователи
		
		
				
		read 		// чтение например с помощью  cat
		w			// запись
		x			// исполнение файла 
				
		x			// для каталога возможность сделать  cd   перейти в нее.
		
		
		rwx   7 
		r-x	   5
		--x   1
