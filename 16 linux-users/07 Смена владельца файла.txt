
	Цель: Научиться менять владельца и группу файлов и папок
	
				man chown			// манул команды   chown
	Меняет пользователя или группу на файле или папке
	
				ls -la				// вывод всех подробных свойств файлов на экран
				
				sudo chown :vagrant .zshrc		// Вызываем команду без пользователя.
												// Здесь у файла .zshrc  поменяется группа на vagrant
				
				sudo chown -R :vagrant .local/		// Флаг  -R  рекурсия меняет содержимое папки
				
				
				
ТЕСТЫ
	Следующая команда должна сменить владельца файла 'test' на 'root', а группу на 'administrators'. 
	Впишите пропущенное слово.
		sudo chown root:administrators test
		
	Следующая команда сменит владельца файла 'test' на 'root'. Группа при этом меняться не должна. 
	 Впишите пропущенное слово. (Используйте самый простой из возможных синтаксисов, без использования ":")
		sudo chown root test
		
	Следующая команда сменит группу файла 'test' на 'supergroup'. Впишите пропущенное слово.
		sudo chown :supergroup test
		
	Следующая команда сменит группу всех файлов папки 'dance' на 'supergroup'. Впишите пропущенное слово.
		sudo chown -R :supergroup dance
