
	Цель: Узнать что на самом деле происходит при создании пользователя
	
	Учетные записи пользователей  Linux  находятся в обычном текстовом файле  /etc/passwd.
	
						cat /etc/passwd			// распечатываем содержимое
						
	Каждая строчка в этом файле это отдельный пользователь.
		vagrant:x:1000:999::/home/vagrant:/bin/bash
	
	Также там будет много системных пользователей (как правило uid  и  gid   до 100).
	 Нужны для запуска из под них программ. Это системные пользователи. Под ними нельзя авторизоваться в ОС.
		list:x:38:38:Mailing list manager:/var/list:/usr/sbin/nologin

ТЕСТЫ
	Где находятся учетные записи пользователей Linux?
		в текстовом файле /etc/passwd
		
	Вот пример записи из файла /etc/passwd:
	jsmith:x:1001:1000:Joe Smith,Room 1007,(234)555-8910,(234)555-0044,email:/home/jsmith:/bin/sh
	Какая часть соответствует домашней директории пользователя?
		/home/jsmith
