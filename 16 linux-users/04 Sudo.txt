
	Цель: Разобраться с тем как же правильно выполнять команды от имени другого пользователя 
	 и получать повышенные привилегии
	 
	
					sudo ls   // так использовать, он спросит пароль Суперпользователя
					
					
	Конфигурация sudo   находится в файле      etc/sudoers   
	у обычного пользователя нет прав на этот конфиг
					
					sudo cat etc/sudoers   			// вывод содержимого файла
					
	sudo  просит пароль, а потом 5 минут его не запрашивает.

					sudo -i				// интерактивный режим
	
					sudo visudo   		// запуск редактора файла   sudoers  
					
					sudoedit vim 		// доступ к редактору sudo 
					
					man sudo			// справка по команде  sudo
