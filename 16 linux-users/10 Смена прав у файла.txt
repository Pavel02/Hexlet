
	Цель: Научиться менять права у папок и файлов
	
			chmod 777 file		// изменение прав на файл file 
			chmod a-r file		// для всех групп убрали чтени файла
			chmod u+r file		// для владельца возврат чтения файла
			
			chmod o= file		// Отбираем все возможные права у остальных групп

chmod o-rwx file
Когда после категории пользователей (в данном случае o) ставится + или - - то сброса не происходит, 
происходит изменение текущего состояния - т.е. назначаются или удаляются только указанные в команде 
права доступа, значения остальных (не указанных в команде) прав доступ остаются неизменными.

Когда после категории пользователей (u, g или o) ставится = - то происходит "сброс" ранее установленных 
прав - т.е. вместо них назначаются указанные в команде права доступа, а значения остальных (не указанных 
в команде) прав доступа обнуляются.

Обновление состояния:

-rwxrwxrwx file.txt
chmod o-wx file.txt
-rwxrwxr-- file.txt

Сброс:

-rwxrwxrwx file.txt
chmod o=r file.txt
-rwxrwxr-- file.txt

Если после = ничего нет (пустая строка), то все права доступа для указанной категории пользователей будут удалены:

-rwxrwxrwx file.txt
chmod o= file.txt
-rwxrwx--- file.txt



ТЕСТЫ
	Допишите команду так, чтобы участники группы больше не могли редактировать этот файл.
		chmod g-w file
		
	Допишите команду так, чтобы права на файл для others стали 0. Используйте сброс, вместо изменения относительно текущего состояния.
		chmod o= file
