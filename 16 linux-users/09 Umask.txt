	
	Цель: Понять как назначаются права папкам и файлам при создании и как на это можно влиять.
	
				mkdir directory		// Создается каталог  directory   с правами   drwxr-xr-x 		или в цифре  755
				touch file			// Создается файл   file  с правами по умолчанию   -rw-r--r--     в цифре   644
				
				
				umask				// выводит   0022
				
	По умолчанию  базовыми правами считаются для директории  777   для файла  666 
	umask При создании файла или папки берут базовые права и из них вычитаются   umask 777 - 022 = 755 и получаются те 
	 права которые будут выданы.
	 
	umask можно поменять чтобы при следующем заходе в систему он стал другим. 
				umask 002
