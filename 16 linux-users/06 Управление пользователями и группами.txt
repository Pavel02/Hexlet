
	Цель: Научиться создавать модифицировать и удалять учетные записи
	
	Стандартные команды для всех Линукс дистрибутивов
		groupadd   groupdel  groupmod groups
			
						sudo groupadd test		// добавление группы  tes 
						sudo groupdel test
						
		useradd 
		
		
		gpasswd			// команда для управления зависимостями между полльзователями и группами
						
						sudo gpasswd -d vagrant docker		// Удаление пользователя vagrant  из  группы  docker  
						sudo gpasswd -a vagrant docker		// Добавления в группу 
		
	Эти команды можно набирать без параметров и по ним будет справка.
	
	В дистрибутивах Debian имеются команды   adduser   addgroup    delluser 
						adduser   		// автоматом создается  user  с настройкам по умолчанию
						
						
						passwd 			// замена пароля пользователю

ТЕСТЫ
	Напишите команду, выводящую на экран список групп, в которых состоит текущий пользователь
		groups
		
	Следующая команда удаляет пользователя 'hexus' из группы 'hexlet'. Допишите недостающую часть команды.
		sudo gpasswd -d hexus hexlet
		
	Следующая команда добавляет пользователя 'hexus' в группу 'hexlet'. Допишите недостающую часть команды.
		sudo gpasswd -a hexus hexlet

		groups x
