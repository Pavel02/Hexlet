
	Цель: Понять кто такой root и зачем он нужен
	
	Суперпользователь  root  c неограниченными правами в системе вплоть до полного удаления ОС.
	 Поэтому под рутом никто не работает, а использует его по необходимости выполнения административных задач.
		Потому что настраивать систему и устанавливать ПО может только  root.
	
	root  может быть только один. 
	
	У root   uid = 0 gid = 0 . 
	
	при этом имя может быть и не root, ОС при проверке прав пользователя проверяет именно  uid    и  gid. 
	
	Домашняя директория суперпользователя  находится в корне и называется /root
