
	Установка в Win
		choco install -y vcredist2015		// Системная библиотека, нужная для запуска кода.
		choco install -y php atom			// интерпретатор РНР и редактор текста Атом
		
	PHP сам по себе обычная программа с консольным интерфейсом.
	И у РНР  есть исполняемый файл с именем php . Этому файлу, как аргумент
	 надо передавать название файла (путь до него), и тогда он его исполнит и сделает, что там написано.
		
	Запуск РНР в командном режиме.
		В редакторе создаете файл с расширением php	(например в атоме).
		В каталоге,  например, с этим же файлом запускаем   babun  через контекстное меню.
			ls					// смотри файлы в данном каталоге.
			php index.php		// открываем файл интерпретатором
		
	В редакторе атом можно установить linter-php  пакет для проверки синкаксиса
	Также есть дополнения, проверяющие стандарты кодирования PSR.
	
	
	Установка на Ubuntu
		sudo add-apt-repository ppa:ondrej/php
		sudo apt-get update
		sudo apt-get install php7.0
		
	Установка на Mac
		brew install homebrew/php/php70
		brew install homebrew/php/php70-xdebug
		
		
	Утилита для проверки стандартов кодирования	
			https://github.com/squizlabs/PHP_CodeSniffer
	Пакет для атом, интегрирующий утилиту выше, для проверки стандартов. 
			https://atom.io/packages/linter-phpcs
			
	Если не получается установить atom из пакетов, можно попробовать скачать его с официального сайта		
		https://atom.io/
			
			
			
	Атом для ubuntu
		sudo add-apt-repository ppa:webupd8team/atom
		sudo apt-get update
		sudo apt-get install atom
