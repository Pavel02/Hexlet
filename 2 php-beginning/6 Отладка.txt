
	Что такое отладка? 
		Процесс поиска и устранения ошибок.
			Большую часть времени именно отладкой занимается программист (а не написанием кода).
			
			
	Синтаксическая ошибка (срответствие грамматике языка)
		Возникает во время парсинга.
			Parse error: syntax error, unexpected ';' in .....
		
	
	PHP errrors
		Notice
		Warning
		Fatal	
	Для профф разработки нельзя допускать Notice и Warning.
		
		
	Stack Trace.
		Показывает путь выполнения программы в коде.
		Предоставляет информацию по файлам и коду (с указанием строки), 
		  который был задействован при работе программы
		Формируется набор вызовов. По которым нужно отследить последовательность вызовов в программе.
		
		
	Xdebug  -  https://xdebug.org
		Расширение php  для упрощения и улучшения отладки.
		На офф сайте описание, как устанавливать, потому что там конкретная версия, для конкретной вер PHP
			Имеется возможность удаленной отладки.
			Добавляет  stack trace  для всех видов ошибок (по умолчанию для Notice и Warning   stack trace отключен).
			Позволяет превратить Notice и Warning в Fatal ошибки. (они перестанут игнорироваться).
