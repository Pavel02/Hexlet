
	DQL - подмножество  SQL отвечающее за выборку данных.
	Содержит одно выражение, начинающееся со слова  SELECT. 
	Мы изучаем стандарт SQL, поэтому изученное будет работать во всех реляционных БД.
	
					psql -U postgres			// в терминале входим в СУБД PostgreSQL

					\c test						// Подключение к БД с именем  test
					
					\d							// Просмотр имеющихся таблиц в текущей БД
					
					select * from products;		// Выбор всех полей из таблицы  poducts
					
					select price, name from products; 		// Можем выбрать любые поля в нужном порядке
					
					select price, name AS name2 from poducts;	// Часто нужно не просто выбрать поле но и переименовать его
																// Например это используется в ORM, чтобы понимать к какому объекту,
																// какая колонка относится, когда делается выборка связанных объектов
																//  Также когда идет выборка из разных таблиц и нужно одинаковые имена различать между собой
	
	SQL может выполнять арифмитические операции
					select price + 2 from products;			// К каждому значению будет прибавлено 2
					
	Как раз для случае будет удобен  alias (псевдоним)		
					select price + 2 as price2 from products;
					
	Можем использовать SQL  как калькулятор 
					select 2 + 3; 							// выведет цифру 5
					
	Есть встроенные функции
					select random() + price from poducts;	// К цене будет прибавлено рандомное число
					select random();						// Просто рандомное значение
