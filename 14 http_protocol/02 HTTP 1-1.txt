
	HTTP/1.1    расширяет предыдущий  1.0  и добавляет виртуальные хосты,которые являются неотемлемым атрибутом
	
	В HTTP/1.0    было достаточно только    request line    HEAD / HTTP/1.0    
	В HTTP/1.1 появился   host:    указывает какой домен должен отдаться с этого ip
								
								GET / HTTP/1.1
								host: hexlet.io
								
	В HTTP/1.1   соединение по умолчанию не закрывают и можно продолжать работать.
	для явного закрытия делаем запрос с заголоком 
								GET / HTTP/1.1
								host: hexlet.io
								connection: close
