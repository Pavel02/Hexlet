
	Цель: Рассмотреть полезные и положительные аспекты сохранения состояния.
	
	
	
ТЕСТЫ
	Возможно ли реализовать подобную функцию для генерации случайных чисел без использования 
	 замыкания и без сохранения состояния?
			да, но придется передавать как минимум один аргумент вручную при каждом новом вызове
	
	Функция random из видео возвращает
		функцию, возвращающую случайное число	
	
	В описанном на видео генераторе случайных чисел происходит сохранение состояния с помощью замыкания. 
	 Что именно там сохраняется таким образом?
		число, являющееся базовым значением для вычисления следующего случайного числа
	
	
	
ЗАДАНИЕ
	file: Solution.php
	Измените функцию random из видео так, чтобы можно было обнулять сгенерированную последовательность.

	Пример:

	$seq = random(3);
	$result = $seq(); // 22
	$seq(); // 6
	$seq(); // 23

	$seq("reset");

	$result == $seq(); // 22
	
							 // Решение
								<?php

								namespace App\Solution;

								// BEGIN (write your solution here)
								function random($seed)
								{
									$init = $seed;
									return function ($method = null) use (&$seed, $init) {
										$a = 45;
										$c = 21;
										$m = 67;

										switch ($method) {
											case "reset":
												$seed = $init;
												break;
											default:
												$seed = ($a * $seed + $c) % $m;
												break;
										}

										return $seed;
									};
								}
								// END
