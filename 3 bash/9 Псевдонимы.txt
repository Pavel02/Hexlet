
	Псевдонимы. они же alias. Для очень часто используемых комманд можем назначить короткие коды.
	
			alias				// Увидим спсиок всех псевдонимов, имеющихся в системе.
			alias c='cd ..'		//  Задали псевдоним  с  переход в каталог выше
			
			unalias с			// Удалит псевдоним  с  из системы
			
			type ll				// расшифрует содержимое команды
