
	Программа  ls  выводит данные в стандартный поток вывода  STDOUT
	А  уже  BASH выводит на экран.
	
			ls > output			// В файле output  окажется вывод команды  ls
			cat output			// Просмотр содержимого файла  output
			
	Можем считать файл для этого используется стандартный поток ввода  STDIN
	
			cat unsorted
			sort < unsorted		// сортируем содержимое файла  usorted  считывая данные прямо из файла.
			sort < unsorted > sorted		// сортируем содержимое файла  usorted а результат не на экран, а в файл sorted
			cat sorted			
			
	Конвейер   |    перенаправляет STDOUT одного процесса в  STDIN  другого.
	 и позволяет строить цепочки из программ.
			cat unsorted | sort		// Вывод перенаправили с экрана в команду  sort
			cat unsorted | sort | uniq		
			
	Конвеер часто используется когда нужна часть файлов
			ls | grep test			// На экран выведутся файлы содержащ в названии test
			ls | grep test | grep rkt		// И можем еще дальше продолжать 
			
			ls | wc			// Покажет количество слов, строк и букв, которые будут в выводе команды ls
			
			
			
 ЗАДАНИЕ 
	Выведите на экран количество уникальных животных в названии которых присутствует подстрока American.
	Список животных лежит в файле  source. 
		Для подсчета количества строк изпользуйте   wc -1
		uniq  убирает дубли, если они рядом, а не в разброс. 
		Чтобы этого добиться используйте   sort.
		
			cat source | grep American | sort | uniq | wc -l
