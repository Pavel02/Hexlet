
	При запуске какой-либо программы в BASH, вначале оболочка находит исполняемый файл этой программы.
	
			ls			// Не встроен в BASH
			wich ls		// Увидим, что конкретно будет запущено
			
			env			// Просмотр списка переменных окружения текущей сессии
			
	Переменная окружения PATH - содержит список директорий с исполняемым файлом различных программ
	Переменные окружения можно использовать в программах.
