		pwd 	// Путь к текущему каталогу
	
	В Юникс системах нет понятия локальный диск. Любой носитель будет отображаться как каталог.
	При этом в Юникс системах все есть файл! А привычный каталог это тоже файл просто со 		 списком файлов. 
	..  это жесткая ссылка на родительскую директорию.
	.   это текущая директория.

		ls 	// Список файлов в текущем каталоге
		ls projects	// посмотрим что находится в каталоге projects, но 
				    без перехода в нее
		change directory
		cd projects	// Переход в каталог с обозначенным именем.
		cd /		// переход в корень диска
		cd ..		// переход в каталог выше
		cd		// без аргументов это переход в домашнюю директорию 
		cd ~/projects	//
		~ 		// Это обозначение домашней директории
		tab + tab 	// Показ имеющихся файлов при наборе команды cd ~/
